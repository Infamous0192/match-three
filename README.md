Ini adalah game Match Three. Game ini dibuat menggunakan Engine Unity 2D. Game saya buat sebagai submisi tugas saya di Agate Academy.

## Getting Started
Untuk Memulai permainan anda dapat men-download atau meng-clone repository ini. Lalu anda dapat meng-extract file ./Build.zip yang ada pada directory root repository ini. Selanjutnya untuk memainkannya bisa dengan membuka file "Match Three.exe", kemudian tekan "Play".
